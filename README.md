# Gestore Zaino Scolastico (Python)

Questo progetto è un'applicazione console per la gestione delle materie scolastiche, sviluppata per esercitare i concetti di **Programmazione a Oggetti (OOP)**.

## Caratteristiche
- **Persistenza dati**: Le materie vengono salvate automaticamente in un file `.txt`.
- **Sincronizzazione**: Gestione dinamica tra la lista in RAM e il database su disco.
- **Robustezza**: Utilizzo della libreria `os` per la gestione intelligente dei percorsi file.
- **Validazione**: Controllo della presenza di una materia prima della rimozione.
